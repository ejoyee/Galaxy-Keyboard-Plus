# 2. MCP란?

**MCP (Model Context Protocol)**는  
AI가 실제로 도구를 실행하게 해주는 기술입니다.

🔧 연결 가능한 도구 예시:
- Notion  
- Gmail  
- File System  
- Google Drive  
- Google Sheet  
- Discord  
- Google Calendar  
- PDF 파일  
- Excel 문서

---

👤 **페르소나: 비개발자**  
- 주 업무: 문서 작업, 일정 관리  
- 기술에는 익숙하지 않음  
- 코딩, 설치, 설정은 어려움  
- 하지만 자동화에 대한 니즈는 높음
